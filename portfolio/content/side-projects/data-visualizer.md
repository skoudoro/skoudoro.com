An experimental web application for exploring high-dimensional scientific data through interactive charts and real-time analysis. Built to scratch an itch I had when working with neuroimaging datasets that were awkward to visualize in standard tools.

It handled volumetric data, fiber tractography, and time series from fMRI experiments. The rendering pipeline was built on top of FURY, the open-source visualization engine I help maintain.

Shelved in favor of contributing those improvements directly to FURY and DIPY rather than maintaining a separate app.
