A platform designed to help researchers across different institutions find collaborators, share datasets, and coordinate on neuroimaging projects.

The pain point it addresses: academic collaborations often start informally and collapse due to lack of shared tooling. This tool provides a lightweight space to post research interests, signal dataset availability, and bootstrap coordination before committing to a full collaboration agreement.

Still a work in progress — the core matching logic is built but the UI needs more polish before a public launch.
