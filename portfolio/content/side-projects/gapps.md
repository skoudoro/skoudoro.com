A Python library that lets developers build add-ons for Google Workspace with minimal boilerplate. Supports Gmail, Google Chat, Calendar, Sheets, Drive, Docs, and other Workspace apps.

The goal was to make it easy to create powerful automations and integrations without having to wrestle with the raw Google APIs — just write Python and ship.
